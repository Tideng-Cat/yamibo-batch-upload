# yamibo-batch-upload

**注意：所有代码都是LLM写的，使用此脚本风险自负。**

使用方法：

0. 准备好用户脚本管理器。参考[这个页面](https://greasyfork.org/zh-CN)上的第1步。（注意：随意安装互联网上的脚本有各种风险，包括账号和个人信息被窃取。）
1. 点击 https://github.com/Tideng-Cat/yamibo-batch-upload/raw/refs/heads/main/yamibo-batch-upload.user.js 。然后点安装。
2. 打开百合会发帖页面
<img width="500" src="https://github.com/user-attachments/assets/cecf5d16-8230-4ba7-8009-90ea2d5bee49" />

3. 点Choose Files，选择要上传的文件。你可以多次点Choose Files上传（文件不防重复，你需要自己检查）。完成此步后，界面如图，所有图片都应出现在第二列。
<img width="500" src="https://github.com/user-attachments/assets/b088a14d-23ef-40e3-8758-671d668fd29b" />

4. 点Sort by Original Filename按文件名排序全部图片，或点击Top移到最顶/Bottom移到最底/Up上移一位/Down下移一位。完成此步后，第二列应符合你想要的顺序。
5. 点第三列下面的Copy Tags，粘贴到编辑框。
<img width="500" src="https://github.com/user-attachments/assets/7c78edfe-fe21-48e7-8a04-183ffcd8fd0b" />

6. 勾选右上角纯文本，再取消勾选。（这一步好像不是必要的）
<img width="200" src="https://github.com/user-attachments/assets/e6955191-535c-4b4d-ab6e-de6189734eba" />
